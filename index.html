Smart Brain

Smart Brain is a full-stack web application that allows users to detect faces in images using AI-powered face recognition (Clarifai API). Users can register, sign in, and track the number of images they've submitted. The frontend is built with React.js, while the backend uses Node.js and Express, connected to a PostgreSQL database for storing user data and activity. -
